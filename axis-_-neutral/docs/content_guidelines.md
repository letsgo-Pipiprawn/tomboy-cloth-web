# AXIS / NEUTRAL 内容生成规则（Content Guidelines）

> 目的：确保日常内容输出（TikTok / Instagram / Blog / 商品文案）语气统一，可直接执行。  
> 默认市场：Australia（AU）。

## 1. 品牌语气与写作原则

## Tone
- Direct（直接）
- Calm（克制）
- Confident（有态度）

## 写作原则
- 句子短，信息密度高。
- 先讲轮廓、比例、场景，再讲情绪。
- 少形容词堆砌，不写空泛“高级感”。
- 永远避免“讨好式女性叙事”。

## 推荐词汇
- `structured`
- `clean shoulder`
- `wide leg`
- `borrowed lines`
- `city-ready`
- `androgynous`
- `tomboy aesthetic`

## 禁用词汇（硬规则）
- `cute`
- `girly`
- `flattering`
- `pretty little`
- `perfect body`
- `fix your shape`
- `must buy now`
- `last chance!!!`

## 慎用词汇（需证据）
- `best`
- `premium`
- `luxury`
- `guaranteed`

---

## 2. Caption 固定结构（必用）

每条 caption 按以下结构写：

1) Hook（第一句抓人）  
2) Product / Fit（产品与版型信息）  
3) Context（穿着场景）  
4) CTA（明确下一步动作）

### 标准模板

```text
[Hook]
[产品名 + 1个关键版型点]
[场景句：where/when/how]
[CTA]
```

### 示例

```text
Borrowed lines, owned attitude.
The Oversized Charcoal Blazer with a clean shoulder and relaxed drape.
Made for weekday meetings and late-night city runs.
Tap to shop the drop.
```

## 3. TikTok 文案模板

## 模板 A：穿搭展示（15s-25s）
```text
Hook: Not your usual womenswear silhouette.
Body: Boxy shoulder. Wide leg. Zero overthinking.
CTA: Watch till the end, then shop via link in bio.
```

## 模板 B：版型教育
```text
Hook: Oversized is not the same as sloppy.
Body: We build shape through shoulder, chest, and length balance.
CTA: Save this fit check and compare with your blazer.
```

## 模板 C：新品 Drop
```text
Hook: Drop 01 is live.
Body: 3 core pieces. 1 clear attitude.
CTA: Limited preorder now open.
```

## 4. Instagram 文案模板

## 模板 A：Feed（品牌向）
```text
[Hook]
[产品信息：面料/版型/细节]
[情境句]
[CTA + 轻量标签]
```

示例：
```text
Cut with intent. Worn your way.
The Utility Jacket is built with a relaxed torso and clean line sleeve.
For city mornings, gallery afternoons, and after-dark plans.
Shop via bio. #tomboyfashion #australianfashion
```

## 模板 B：Reel（转化向）
```text
Hook: One silhouette, three settings.
Body: Office / Street / Night, same core pieces.
CTA: Comment "fit" for size guide link.
```

## 模板 C：Story（互动向）
```text
Poll: Boxy shoulder or relaxed shoulder?
Follow-up: We are dropping both in next batch.
CTA: Vote now + turn on drop alerts.
```

## 5. 场景化内容清单（每日可选题）

- `Monday Fit`：通勤搭配
- `Fit Check`：肩宽/胸围/衣长对比
- `1 Piece 3 Looks`：单品复用
- `Detail Close-up`：面料、走线、口袋
- `Drop Note`：预告 + 时间 + CTA

---

## 6. 输出前检查清单（发布前 30 秒）

- 是否出现禁用词（cute/girly/flattering）？
- 第一行是否有 Hook？
- 是否讲清产品信息（至少 1 个版型点）？
- CTA 是否明确（shop/save/comment/link in bio）？
- 语气是否“直接、克制、有态度”？
