# AXIS / NEUTRAL — 产品选款评审标准

> 给Cursor用。每次看到新产品图，按这个标准打分，满分100分，75分以上直接上架。

---

## 品牌定位（评分时始终对照）

- 品牌名：AXIS / NEUTRAL
- 定位：Cinematic Tailoring，墨尔本中性风女装
- 目标用户：androgynous / alpha female 审美女性，21-40岁，澳洲市场
- 核心词：中性、oversize、有设计感、不俗气

---

## 评分维度（满分100）

### 1. 品牌契合度（30分）

| 问题 | 是 | 否 |
|------|----|----|
| 是否中性/androgynous风格？ | +10 | 0 |
| 是否oversize或宽松版型？ | +10 | 0 |
| 颜色是否在品牌色系内？（黑/灰/深蓝/米/焦糖，避免粉红/荧光） | +10 | 0 |

**直接淘汰条件：**
- 有蕾丝、蝴蝶结、明显女性化装饰 → 0分淘汰
- 格纹英伦奶奶风 → 0分淘汰
- 颜色过于鲜艳（红/粉/黄） → 0分淘汰

---

### 2. 设计细节（25分）

| 问题 | 分值 |
|------|------|
| 有设计亮点（链条/特殊扣子/不对称/机能细节等） | +15 |
| 版型干净利落，没有多余装饰 | +10 |
| 普通无亮点 | +5 |
| 设计细节廉价感（塑料扣/劣质拉链可见） | 0 |

---

### 3. 可售性（25分）

| 问题 | 分值 |
|------|------|
| 是否四季可穿或至少两季？ | +10 |
| 是否好搭配（百搭款）？ | +8 |
| 是否适合澳洲气候？（避免过厚羽绒/过薄透视） | +7 |

---

### 4. 商家图质量（20分）

| 情况 | 分值 |
|------|------|
| 平铺图清晰，白/灰背景，细节看得清 | +20 |
| 有模特上身图，背景干净 | +15 |
| 模特图但背景杂乱 | +8 |
| 图片模糊或角度差 | 0 |

> 注：图质量影响AI换模特的成功率
