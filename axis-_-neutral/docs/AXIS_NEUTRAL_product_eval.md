## 第一关：直男测试（必过，否则直接淘汰）

Lance 是直男，没有时尚背景。这个测试用你的**第一反应**做过滤器。

问自己一个问题：

> **「这件衣服，我第一眼会觉得『这不是男装吗』或者『这不是工作服吗』？」**

| 反应 | 结果 |
|------|------|
| 是的，第一眼像男装 / 工装 / 制服 | ✅ 进入评分 |
| 不确定，但感觉有点特别 | ✅ 进入评分 |
| 第一眼就是普通女装 | ❌ **直接淘汰，不用评分** |
| 第一眼就是淑女 / 甜美风 | ❌ **直接淘汰，不用评分** |

---

## 第二关：硬性淘汰条件（任何一条触发即淘汰）

- 有蕾丝、蝴蝶结、荷叶边、任何明显女性化装饰
- 格纹英伦奶奶风
- **颜色**：粉红、荧光、大红、明黄
- **版型**：收腰、显曲线、紧身
- 材质看起来像睡衣或家居服
- 有大面积印花 / 卡通图案

## 评分维度（满分 100）

### 1. 「咦」因子 — 有没有让人多看一眼的点（25 分）

v2 **核心维度**。好产品要让人滑到这张图时停下来。

| 情况 | 分值 |
|------|------|
| 有一个强设计点：链条 / 不对称 / 夸张翻领 / 机能口袋 / 特殊面料 / 解构细节 | **25** |
| 版型比较特别，但没有明显设计点 | **15** |
| 干净利落但普通，没有记忆点 | **8** |
| 完全没有亮点 | **0** |

**参考「咦」的例子：**

| 例子 | 分 |
|------|-----|
| 西装领口有隐藏链条 | 25 |
| 双排扣但扣子是金属质感 | 20 |
| 极度 oversize 肩线 | 18 |
| 普通黑色单排扣西装 | 8 |

---

### 2. 品牌契合度（25 分）

| 问题 | 分值 |
|------|------|
| 是否中性 / androgynous 风格，不像传统女装 | **+10** |
| Oversize 或宽松版型（肩线宽、腰不收） | **+8** |
| 颜色在品牌色系内：黑 / 深灰 / 浅灰 / 深蓝 / 米 / 焦糖 / 墨绿 | **+7** |

---

### 3. 可售性（25 分）

| 问题 | 分值 |
|------|------|
| 四季可穿或至少两季（澳洲秋冬春） | **+10** |
| 好搭配，买家能自己组合造型 | **+8** |
| 适合澳洲气候，不过厚不过薄 | **+7** |

---

### 4. 商家图质量（25 分）

| 情况 | 分值 |
|------|------|
| 平铺图清晰，白 / 灰 / 黑背景，细节看得清 | **25** |
| 模特上身图，背景干净，角度好 | **20** |
| 模特图但背景杂乱 | **10** |
| 图片模糊或角度差 | **0** |

> 图质量直接影响 AI 换模特成功率。**低于 10 分**建议联系供应商要新图。

## 评分结果

| 分数 | 决策 |
|------|------|
| **85–100** | ⭐ **主推款**，优先 preorder，做 AI 模特大图 |
| **75–84** | ✅ **上架**，正常推广 |
| **60–74** | ⚠️ **备选**，等主推款跑出数据再决定 |
| **60 以下** | ❌ **淘汰** |
