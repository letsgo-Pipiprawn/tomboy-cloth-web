# AXIS / NEUTRAL 供应商沟通模板（Supplier Comms）

> 目标：标准化和 1688/工厂沟通，减少信息遗漏，提高打样与大货成功率。  
> 适用：找款、打样、报价、改版、下单、质检。

## 1. 首次联系 1688 商家（开场模板）

```text
你好，我们是澳洲女装品牌 AXIS / NEUTRAL，长期做 androgynous / 男装剪裁女装（city tailoring，中性色）。
我们正在寻找可长期合作的工厂，主要品类是：
1) Oversized blazer
2) Wide-leg trousers
3) Utility jacket / shirt

请先提供以下信息：
1. 工厂还是贸易公司
2. 可做的最小起订量（MOQ）
3. 打样费和打样周期
4. 大货周期（首单/补单）
5. 面料来源与可选克重
6. 是否支持我方版型和吊牌包装
7. 过往同类产品图片/视频
```

## 3. 打样要求清单（必须发给工厂）

## 基础信息
- 款号 / 产品名
- 目标版型（boxy / relaxed / straight）
- 目标客群与穿着场景

## 技术要求（Tech Pack 最低项）
- 尺寸表：胸围、肩宽、衣长、袖长、裤长、腰臀
- 公差（Tolerance）：建议 ±0.5cm（关键位）
- 面料成分、克重、手感要求
- 辅料要求：拉链、扣子、里布、衬布
- 车线要求：线迹密度、明线位置、加固点
- 洗标 / 吊牌 / 包装规范

## 样衣交付要求
- 样衣照片（正/侧/背/细节）
- 上身视频（静态 + 行走）
- 尺寸实测表（逐项）
- 样衣问题备注（工厂自检）

## 6. 质检标准（QC）- 简版

## A. 尺寸
- 关键尺寸必须在公差内（建议 ±0.5cm / 非关键 ±1cm）
- 任一关键位超差比例 >5%，判定复检

## B. 外观
- 无明显污渍、破损、色差异常
- 对称部位误差可控（领、门襟、口袋、裤缝）

## C. 做工
- 无跳线、断线、爆线
- 高受力位需有加固
- 线头清理完整

## D. 功能
- 拉链顺滑、纽扣牢固
- 洗后尺寸变化在可接受范围

## E. 包装
- 吊牌、洗标、条码准确
- 外袋无脏污、装箱数量正确

## 8. 供应商评分卡（每单复盘）

每次合作后按 1-5 分打分：
- 报价透明度
- 打样准确率
- 改版配合度
- 交期稳定性
- 大货一致性
- 沟通响应速度

## 处理规则
- 平均 >=4：优先长期合作
- 3-4：保留但需观察
- <3：降级或替换
