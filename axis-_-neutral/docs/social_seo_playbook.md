# AXIS / NEUTRAL — Social & SEO Playbook（战术层）

> 配合 Cursor Skill：`marketing-social-advisor`  
> 默认市场：Australia（AU）  
> 更新：按季度回顾发帖节奏与 SEO 目标；下方为启动默认值，可按实际精力调整。

---

## 1) 每周有机内容节奏（建议）

| 渠道 | 频率 | 优先级内容类型 |
|------|------|----------------|
| **TikTok** | 3–5 / 周 | Fit Check、1 Piece 3 Looks、Drop Note |
| **IG Feed** | 2–3 / 周 | 品牌向单品 + 场景图（4:5） |
| **IG Reel** | 2 / 周 | 转化向（Office/Street/Night 三联） |
| **IG Story** | 日常轻量 | Poll / 尺码互动 / Drop 倒计时 |

**本周主题轮换（任选 1 为主轴）：**

- Drop / Preorder 引流
- Fit Check（肩线、衣长、阔腿比例）
- 1 Piece 3 Looks（同一单品多场景）
- Detail Close-up（面料、口袋、走线）

TikTok 优先于 IG Feed（发现流量）；IG 负责沉淀与 bio 转化。

---

## 2) TikTok vs IG 分工

| | TikTok | Instagram |
|--|--------|-----------|
| **目标** | 发现、保存、进 bio | 信任、复访、DM/评论转化 |
| **钩子** | 前 1 秒轮廓冲突 | Feed 第一行态度句 |
| **CTA** | link in bio / save fit check | shop via bio / comment "fit" |
| **标签** | 3–5 个轮换 hashtag | Feed 2–4 个，不过度堆叠 |

## 3) 本月 SEO 焦点（AU）

**Tier 1（页面级优先）：**

1. `androgynous women's clothing australia` — 首页 / About 支撑
2. `women's wide leg trousers australia` — Wide-Leg Black Trouser PDP

**Tier 2（Blog 长尾，每月 1–2 篇）：**

- `gender neutral style australia`
- `tomboy fashion australia`（长尾，不作 Hero H1）

**On-page 规则：**

- `meta_title` ≤60 · 格式 `{Product or Topic} | AXIS / NEUTRAL`
- `meta_description` ≤155 · 1 主词 + 1 长尾 + Australia/Melbourne 自然出现
- Blog：H2 回答具体穿搭问题，内链到对应 `/products/{slug}`

## 6) 反模式（不做）

- 追无关节日挑战 / 幼态滤镜 / “变装可爱”梗
- girl-boss、hustle、dominate、must buy now、last chance!!!
- 用 `tomboy` 作 Hero 主标题或品牌人格句
- 无版型信息的纯氛围 caption
- SEO 关键词堆砌或同一 meta 描述重复全站
